# Code of Conduct

## Our Standard

Interactions in StangaNet spaces (Issues, Discussions, comments) must be respectful and constructive.

Expected behavior:

- Use welcoming and inclusive language
- Be respectful of differing viewpoints
- Accept feedback gracefully
- Focus on what is best for the project and the community

Unacceptable behavior:

- Insults, personal attacks, or derogatory language
- Spam or off-topic content
- Sharing others' private information without consent
- Any conduct that would be considered inappropriate in a professional setting

## Enforcement

The maintainer reserves the right to remove comments, close issues, and block users who violate this standard, without prior warning.

## Scope

This Code of Conduct applies to all project spaces managed by StangaNet on GitHub.

## Contact

If you witness a violation, open a private report via GitHub or contact the maintainer directly through GitHub.
